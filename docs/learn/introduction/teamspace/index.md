# Teamspace

For more details please have a look at the [Teamspace management][DOCSLoopTS] documentation.

## Overview

--8<-- [start:Overview]

A teamspace acts as a logical partition of larger system providing a secure and isolated "environment/workspace" for multiple teams to work independent.

--8<-- [end:Overview]

## Create new Teamspace

--8<-- [start:CreateTeamspace]

| Step | Details                                                                                              | Additional Information                                                  |
|:----:|:-----------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------|
|  0   | If Loop environment has been freshly set up just click on the **Create Teamspace** Button to proceed | ![Create Teamspace][TSInitCreateButton]                                 |
|      | then proceed to step 2, else with next step                                                          |                                                                         |
|  1   | Click on the Teamspace icon ![TeamSpace Icon][TSIcon] to view your Teamspaces or create a new one    | ![Teamspace menu][TSMenue]                                              |
|      | Click on **Create teamspace**                                                                        | ![teamspace menu - Create teamspace Link][TSCreateTSentry]              |
|  2   | The **Create Teamspace** screen is shown                                                             | ![Create Screen - TeamSpace][TSCreateNewScreen]                         |
|      | Select the **Icon/Color** for the TeamSpace                                                          | ![Teamspace - Iconcolor][TSCreateTSIconColor]                           |
|      | Enter the name and a Description and press **Next** ![Next Button][NextButton]                       | ![Teamspace - Create New][TSCreateNewDialog]                            |
|  3   | You can search and **add** Teammembers to a Teamspace                                                | ![Teamspace - Create New - search Member][TSAddTeammember]              |
|      | Teammember is added to a list. Proceed to add more members if needed.                                | ![Teamspace - Create New - Member added][TSTeamMemberAdded]             |
|  4   | Press **Create** to proceed                                                                          | ![Create teamspace Button][TSCreateButton]                              |
|  5   | Teamspace creation will take a few moments, as soon as it finishes a success dialog will be shown    | ![Teamspace - Create New - creation process started][TSCreationStarted] |
|  6   | Teamspace created. Click on **Go to Dashboard** for next steps                                       | ![Teamspace - Create New - creation successfull][TSCreateionFinished]   |

--8<-- [end:CreateTeamspace]

Please proceed with [creating a Loop][GoBackToCreateALoop]
