    AI Integrations

    AI integrations can be setup only with administrative rights.

    AI Integration on Teamspace level
    Switch to the main view of your Teamspace by clicking on DevOps Loop: Loop in the Central App Menu

    View or Create an Integration to an AI Provider

    Step Details Additional Information
    1 Click on Integrations in the Settings** menue for setting up Integrations
    2 A list of integrations is shown if available.
    3 To create a new integration click the Button New Integration
    4 A Dialog for the integration appears
    4.1 Enter a meaningful Name
    4.2 Select the AI Provider
    4.3 Enter your API Key and press Next
    5 Next page of dialog is shown
    5.1 Enter the API endpoint https://api.openai.com/v1
    5.2 Select your prefered model and further configurations
    5.3 click on Save
    6 The newly created AI integration is shown

    DevOps Plan AI integration setup
    Switch to DevOps Plan home view

    Adding an AI User Group

    For easier management of AI access we need to create a Usergroup and add the eligible members to it.

    Step Details Additional Information
    1 Click on User Management in the 3 Dots menue in the sidebar
    2 Click on the Groups tab in the User Management View
    3 Click on Create Group Button to create a new Group
    4 Enter a meaningful name for new Group
    5 Click on the Add User section
    6 Select the users by clicking on the symbol
    7 Click the Add Button to add selected Users to Group
    8 All Groupmemebers are now listed in Group

    Configure Plan AI Integrations

    Step Details Additional Information
    1 Click on Integrations in the Configuration menue in the sidebar
    2 The empty integrations view is shown
    3 Click on New Integration Button
    4 Enter a meaningful name and select your AI Provider and API Key
    5 Press Next Button to enter more Details
    6 Next dialog page opens
    7 Enter API Endpoint if missing https://api.openai.com/v1/chat/completions
    8 Select your prefered model
    9 Set your prefered LLM settings for temperature and more
    10 Click on Save
    11 The new AI integration is created but no Groups are associated
    12 Press the Add Groups Button under Actions Column
    13 Select the new Group you created from above and additional ones you need
    14 Click on ADD Button to save selection
    15 Updated View

    Activate Member Access

    Last step is for every enabled user (from selected groups) is to activate the access

    Step Details Additional Information
    1 Click on your Profile Icon in the Sidebar and select Member Access
    2 A list of integrations available for Member access is shown and their status
    3 Click on the Integrationname to activate access

    Deploy AI Settings

    Step Details Additional Information
    1 Switch to Deploy Home Page
    2 On the Deploy Home page switch to Settings by using the Settings Icon
    3 Scroll down in the System Section to view all sub sections
    4 Click on the AI Settings icon
    5 The AI Integration configuration Dialog opens
    6 In the AI Provider listbox select OpenAI
    7 If AI Provider Endpoint is empty enter this URL https://api.openai.com/
    8 Enter your API Key in the AI Provider API Key field
    9 Click on the Model Name Listbox to view all available Models
    10 Select your prefered moddel from list
    11 Selected Model will be shown
    12 Click on Save to finish setup
